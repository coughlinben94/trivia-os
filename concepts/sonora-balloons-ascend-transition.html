Reduced motion: balloon/wash/outgoing-lift are fully suppressed; round-intro content does a plain 1s opacity fade instead — same pattern as the shiny-transition fix (ANIM-1) already shipped elsewhere in this app.

What this is NOT: the persistent ambient balloons in the background layer never touch this — this is a second, purpose-built balloon living only inside the transition component, since the ambient layer is architecturally locked to never remount.
Self-gate: GPU-only — every animated property above is transform/opacity, blur is static not animated.
